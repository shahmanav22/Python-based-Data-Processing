# Python-Based Data Processing using Pandas

## Objective

This project demonstrates fundamental data processing techniques using Python and Pandas on the Telco Customer Churn dataset.

## Operations Performed

- Data Loading
- Data Inspection
- Missing Value Handling
- Filtering Records
- Sorting Data
- Creating New Features
- GroupBy Operations
- Pivot Tables
- Data Export

## Tools

- Python
- Pandas
- NumPy

## Learning Outcome

This project demonstrates practical data manipulation skills required for preparing datasets before exploratory data analysis and machine learning.
